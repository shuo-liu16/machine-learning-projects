# Machine Learning Projects

这是一个机器学习学习与实践的项目集合，包含入门级项目和系统性的学习笔记。

## 项目结构

```text
machine-learning-projects/
├── beginner/
│   └── Identify-irises/          # 入门项目：鸢尾花分类
├── learning-experience/           # 学习笔记
│   ├── Supervised-Machine-Learning.md
│   ├── Advanced-Learning-Algorithms.md
│   └── Unsupervised-Learning.md
└── README.md
```

## 目录内容

### 1. beginner/Identify-irises（入门项目）

鸢尾花分类是一个经典的机器学习入门项目，使用 KNN（K-近邻）算法对鸢尾花数据集进行分类。

### 2. learning-experience（学习笔记）

系统性的机器学习学习笔记，包含三个核心主题：

#### Supervised-Machine-Learning.md

监督学习笔记，涵盖：

- 机器学习基础概念
- 单变量与多变量线性回归
- 梯度下降算法
- 逻辑回归
- 正则化

#### Advanced-Learning-Algorithms.md

进阶学习笔记，涵盖：

- 神经网络基础
- 前向传播与反向传播
- TensorFlow/Keras 实现
- 从零实现神经网络（NumPy）

#### Unsupervised-Learning.md

无监督学习笔记，涵盖：

- K-means 聚类算法
- 异常检测
- 高斯分布
- 推荐系统基础
- 强化学习入门

## 学习路线

```text
入门阶段
    ↓
监督学习（回归、分类）
    ↓
进阶算法（神经网络）
    ↓
无监督学习（聚类、异常检测）
```

## 备注

- 所有学习笔记使用中文编写
- 入门项目适合机器学习初学者
- 学习笔记适合作为复习参考资料
