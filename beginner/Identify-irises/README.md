# 鸢尾花分类项目

## 1. 项目介绍

### 1.1 项目名称

鸢尾花分类（Identify-irises）

### 1.2 项目目标

本项目旨在使用机器学习技术对鸢尾花数据集进行分类，通过建立模型预测鸢尾花的种类，帮助初学者了解机器学习的基本流程和核心概念。

> ps：这是我的第一个机器学习项目，我想我可以通过这个项目来学习一些机器学习的基本概念和流程。

### 1.3 数据集信息

- **数据集名称**：鸢尾花数据集（Iris Dataset）
- **数据来源**：`data/iris.data`
- **数据规模**：150个样本
- **特征维度**：4个特征
  - 萼片长度（sepal length）
  - 萼片宽度（sepal width）
  - 花瓣长度（petal length）
  - 花瓣宽度（petal width）
- **类别数量**：3个类别
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

### 1.4 运行环境要求

- Python 3.7+
- 依赖包：见 `requirements.txt`

### 1.5 基本使用方法

```bash
# 安装依赖
pip install -r requirements.txt
# 运行模型训练
python train_model.py
# 运行数据可视化
# 需安装 jupyterlab 包
jupyter lab iris_visualization.ipynb
```

### 2.2 为什么选择 KNN 模型？

- 一个样本的类别由它最近的 K 个邻居投票决定。不需要复杂的数学推导，非常适合初学者理解“分类”的核心思想。
- 因为简单直观、无需训练、适合多分类小数据集，是初学者的理想模型。

### 2.3 为什么选择 n_neighbors=3？

- n_neighbors 是 KNN 最重要的超参数，表示考虑多少个最近邻居。
- 避免平局（奇数个邻居
  - K=1 太敏感，容易受噪声影响（单个异常点就会改变预测）。
  - K=2 是偶数，可能产生平局（两个类别各得一票，无法决策）。
  - K=3 是大于1的最小奇数，能有效避免平局，同时保持一定的抗噪能力
- 一个常用法则是 K = √N（N 为样本数）的经验，所以也可以 n_neighbors = 3
